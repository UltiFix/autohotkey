# README #
### How to use, from provided binary###

* Download .exe file from release section
* navigate to startup folder (hit Win+R, type shell:startup hit enter)
* Move the exe there
* double click the exe
* close folder
* in browser navigate to the 1governor.ahk file to see the keyboard shortcut bindings




### How to use, from source (requires AHK)###

* Download/Install AHK Version 1.1.37.02
* Checkout this git repo somewhere
* Right click on 1governor and click send to desktop (create shortcut)
* navigate to startup folder (hit Win+R, type shell:startup hit enter)
* Move the shortcut in there
* double click the shortcut
* close folder
* open the governor file to see hotkeys and functions
